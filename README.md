# OpenCart E-commerce Application - Manual Testing Project

This repository contains a comprehensive manual testing project for the OpenCart E-commerce Application. The project demonstrates systematic testing approaches, test documentation, and quality assurance practices.

## Project Overview

A structured manual testing project following industry best practices to ensure the quality and reliability of the OpenCart e-commerce platform.

## Repository Structure

```plaintext
├── Requirements/
│   └── FRS (OpenCart).pdf
├── Test Planning/
│   ├── Test Plan (OpenCart).pdf
│   └── RTM (Opencart).xlsx
├── Test Design/
│   ├── OpenCart-Test Scenarios.xlsx
│   └── Test cases (Open Cart ).xlsx
└── Test Execution/
```

## Documentation

### 1. Requirements

- Functional Requirements Specification (FRS) document detailing the system requirements

### 2. Test Planning

- Comprehensive Test Plan document outlining the testing strategy and approach
- Requirements Traceability Matrix (RTM) ensuring coverage of all requirements

### 3. Test Design

- Detailed test scenarios covering various aspects of the application
- Test cases with step-by-step instructions for testing procedures

### 4. Test Execution

- Test execution results and reports
- Defect reports and tracking
